# 使用说明

# 直接下载 [BrushClasses.exe](https://github.com/cxfjh/ChaoXingGUI/releases/tag/v1.0.0) 文件运行即可

## 一、脚本运行

1. **克隆或下载项目到本地**
   
2. **安装依赖**

   运行以下命令来安装所需的 Python 库：

   ```bash
   pip install -r requirements.txt
   ```

3. **启动程序**

   运行以下命令启动程序：

   ```bash
   python main.py
   ```
